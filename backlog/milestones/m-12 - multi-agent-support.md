## Description

Support multiple AI coding agents (Claude Code sandboxed/unsandboxed, Codex CLI, Gemini CLI, OpenCode) in both the dashboard (Zellij WASM plugin) and sandbox (bubblewrap wrapper). Includes agent type abstraction, generic command templates, status protocol generalization, per-agent sandbox configs, and individual agent integrations.
