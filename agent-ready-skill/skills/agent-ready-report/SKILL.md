# Agent-Ready Report — Detailed Analysis Document

Generate a comprehensive, human-readable report of the project's agentic readiness assessment.

`$ARGUMENTS` may contain `refresh` to force a new scan, otherwise use existing scores.

For the scoring reference, see `.claude/skills/agent-ready/references/scoring.md`.

## Step 1: Load or Generate Scores

1. Check for `claudedocs/agent-ready-scores.json`
2. If found and `$ARGUMENTS` does not contain `refresh`: use existing scores
3. If not found or `refresh` requested: run the full scan first (invoke `/agent-ready-scan`)

## Step 3: Conversation Summary

After writing the file, display in the conversation:
- Overall score and level
- Path to the full report file
- Top 3 most impactful improvements

## Guidelines

- **Thorough but readable** — use tables, charts, and clear headings
- **Cite evidence** — every score must reference specific files or their absence
- **Be actionable** — every gap must have a concrete fix suggestion
- **Professional tone** — this is a technical assessment document
- **No fluff** — skip marketing language, focus on facts and recommendations
